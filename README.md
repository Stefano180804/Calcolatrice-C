# Calcolatrice in C

Questo progetto è una **calcolatrice semplice in linguaggio C**.  
Permette di eseguire operazioni matematiche di base e avanzate:

- Addizione  
- Sottrazione  
- Moltiplicazione  
- Divisione  
- Potenza  
- Radice quadrata  

Il programma utilizza un **menu numerico** e un ciclo `do-while`, così puoi eseguire più operazioni senza riavviare il programma. Le operazioni sono implementate tramite **funzioni separate**.

---

## 📁 File principali

- `calcolatrice.c` → codice sorgente principale della calcolatrice
